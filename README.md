# 🎬 Movie Library Web Application

A full-stack web application for discovering, organizing and reviewing movies.
Users can search for films, manage personal watchlists, rate movies and interact with other users.

## 🚀 Features

* User authentication (email & Google login)
* Movie search via TMDb API (title, genre, year, actors)
* Personal movie lists: Favorites, Watchlist, Watched
* Ratings (1–10) and user reviews
* Comments on reviews
* User profile with activity history

### Premium features

* Movie recommendation system
* “Cinema mode” – random movie picker
* Data export (PDF / CSV)

## 🧩 Tech Stack

**Frontend**

* React
* Tailwind CSS
* React Router
* Axios

**Backend**

* Node.js + Express
* MongoDB (Mongoose)
* JWT authentication
* TMDb API integration

## 📦 Running locally

**Backend**

```bash
cd backend
npm install
npm run dev
```

**Frontend**

```bash
cd frontend
npm install
npm run dev
```
